# 🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript.

<img width="876" height="998" alt="127 0 0 1_5500_Rock_Paper_scissors_index html" src="https://github.com/user-attachments/assets/8c568436-d273-47f8-9cbc-0c78101c3999" />


## Features

- Play against the computer
- Random computer moves
- Score tracking (Wins, Losses, Ties)
- Persistent score using Local Storage
- Reset score functionality
- Responsive and attractive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
